Typora - Markdown

快捷键

  • opt_cmd+m：最小化
  • opt+cmd+t：表格
  • crtl+m：内敛公式

公式

参考链接：使用Typora添加数学公式

打开Typora选择数学模块

  • 快捷键：crtl+m
  • “$$”+回车
  • 点击“段落”—>“公式块”

以上三种方式都能打开数学公式的编辑如下，所输入的公式居中。
